# Stop-Test

## SYNOPSIS

Stop test case

## SYNTAX

```none
Stop-Test [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Stop-Test writes output to console after test case is done
This function must be called after single test case is done executing

## EXAMPLES

### EXAMPLE 1

```none
Stop-Test
```
